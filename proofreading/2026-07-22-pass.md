# Proof-reading pass — 2026-07-22

Scheduled proof-reading routine, appended to the single rolling proof-reading
PR (#106). One agent per language, **25 random exercises each** (EN + FR),
reviewing both **content** (grammar / spelling / accents / answer correctness /
ambiguity / naturalness) and **structure** (schema conformance + how each item
renders in the engine). A separate adversarial verification agent then
re-checked every edit before commit.

Because this pass reviews the *current* branch state (which already carries all
of #106's earlier fixes), every finding below is net-new by construction — none
re-reports a defect an earlier pass already resolved.

## Changes — 5 items removed, 6 items edited across 10 files

### English

| File | Change | Why |
|---|---|---|
| `used-to-b1.json` | removed 3 items | The `cue:"use"` twins of "He … drink milk", "We … travel by train", "I … call my friends" duplicated the `cue:"drink"/"travel"/"call"` cards. The kept twins follow the file's convention (item 3, `cue:"smoke"`): for the "base verb already in the sentence" pattern, the cue is that following base verb — so the `cue:"use"` copies were the odd ones out. |
| `articles-basic.json` | removed 1 item + fixed 1 | Deleted "{1} children love {2} chocolate." — a near-duplicate of "In general, {1} children love {2} chocolate." with an identical explanation; the "In general," copy was kept because it forces the generic (no-article) reading unambiguously. In that kept item, blank 1's distractor chips were capitalised `["A","The"]` although the blank now sits mid-sentence after "In general," → lowercased to `["a","the"]`. |
| `formal-subjunctive-c2.json` | removed 1 item | "The deadline is unworkable; **be** that as it may, …" was a near-duplicate of "The deadline is unworkable. **Be** that as it may, …" — same carrier sentence, same idiom, same explanation. Kept the sentence-start copy. |
| `present-continuous-a1.json` | edited 1 answer | "I {1} not {2} today" accepted only `["am"]`; the natural contraction `'m` ("I'm not swimming") was missing, inconsistent with the parallel negative item that already accepts `'s`. Added `'m`. |

### French

| File | Change | Why |
|---|---|---|
| `possessive-adjectives-a1.json` | fixed 1 (**high**) | "Avez-vous reçu {1} colis ce matin ?" listed `vos` as a distractor, but « colis » is invariable (un colis / des colis), so both « votre colis » and « vos colis » are correct — two right chips. Replaced the `vos` distractor with `ta` (feminine, 2nd-person singular → unambiguously wrong), leaving « votre » as the only correct answer. Explanation updated. |
| `passe-simple-c1.json` | reworked 1 item | "L'artiste {1} … puis {2} …" used the identical verb pair (`peignit` + `mourut`) and identical "artist finishes final work, then dies" scenario as its neighbour. Re-cast with a different pair — `sculpter → sculpta`, `s'éteindre → s'éteignit` — to remove the duplication. |
| `gerondif-participe-present-b2.json` | edited 1 text | "… en se servant **le** buffet …" is unidiomatic (« se servir » takes « au buffet », not the buffet as a direct object). Changed the carrier to « … au buffet … », matching the item's own explanation ("helping themselves at the buffet"). |
| `nominalisation-c1.json` | edited 1 hint | The prompt hint for `adhérer → adhésion` read « en « -ence/-ion » », but the word follows the `-sion` pattern; « -ence » was a red herring. Narrowed the hint to « -sion ». |
| `concordance-des-temps-c2.json` | edited 1 answer | "…s'il avait insisté davantage, ils {1} sa proposition." accepted only « auraient accepté »; in this C2 file (which elsewhere lists literary alternants) the conditionnel passé 2ᵉ forme « eussent accepté » is equally correct. Added it. |
| `partitive-articles-a2.json` | edited 1 text | "Tu dépenses {1} argent pour le voyage." near-duplicated "Tu as {1} argent pour le voyage." (same noun, same tail). Reworded the tail to « chaque week-end » to differentiate; answer « de l' » unchanged. |
