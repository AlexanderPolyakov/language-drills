# Proof-reading pass — 2026-07-16

A content + structure proof-reading pass over a random sample of exercises in
both languages, plus verification of the currently-open proof-reading PRs.

## Method

- One proof-reader per language, each given **25 randomly selected exercise
  files** (50 files total, ~53% of the EN corpus and ~61% of the FR corpus).
- Each file was checked for **structure** (schema conformance via
  `scripts`-style validation: required top-level fields; `type ∈ {cloze, fill,
  transform}`; `{n}` placeholders matching `blanks[]`; every cloze `answer`
  present in its `options`; no duplicate options/items; no doubled-word
  rendering) and **content** (grammar, answer-key correctness and uniqueness,
  explanation/hint/cue consistency, spelling, accents/elision for French).
- A separate verifier independently re-checked every change in each open
  proof-reading PR against the schema and the language rules.

## Result: no net-new defects

Both samples passed cleanly — **0 structural issues and 0 content defects** in
the 50 files reviewed. Two French candidates were investigated and correctly
dismissed as non-defects:

- **Elision in `fill` items** (e.g. `Je {1}` → `aurai`, `qu'il ne {1}` →
  `ait/eût`): the `fill` renderer always inserts the parenthetical `(cue)`
  between the preceding word and the input box, so the fixed word and the
  answer are never adjacent — the answer strings themselves are correctly
  spelled.
- **`Nous {1}` → `nous levons`** in `reflexive-verbs-a2`: renders as `Nous nous
  levons`, which is the required subject + reflexive pronoun, not a doubled
  word.
